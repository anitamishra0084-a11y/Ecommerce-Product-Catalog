# 🛒 ShopEase - E-Commerce Product Catalog

A responsive and interactive **E-Commerce Product Catalog** built using **HTML, CSS, and JavaScript**. The project provides a modern shopping experience with product browsing, shopping cart functionality, local storage, and a clean user interface.

## 🚀 Features

* Responsive design for desktop and mobile
* Modern homepage with hero section
* Product catalog with attractive cards
* Search and category filter
* Shopping cart with Local Storage
* Quantity management
* About page
* Contact form with JavaScript validation
* Customer testimonials
* Smooth animations and hover effects

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage

## ▶️ How to Run

1. Download or clone the repository.
2. Open the project folder in VS Code.
3. Open `index.html` in your browser using Live Server or any modern browser.
